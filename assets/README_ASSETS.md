# Assets de WarmiNet

Esta carpeta contiene todos los recursos multimedia de la aplicación.

## Instrucciones para completar los assets:

### 1. Logo (assets/logo/)
Crear los siguientes archivos con el logo de WarmiNet en colores morado (#4b135f) y rosa (#fd71b2):

- `icon.png` - 1024x1024px - Ícono principal
- `splash.png` - 1284x2778px - Pantalla de inicio
- `adaptive-icon.png` - 1024x1024px - Ícono adaptativo para Android
- `favicon.png` - 48x48px - Favicon para web
- `logo.png` - Imagen del logo para usar en la app (transparente, ~512x512px)

### 2. Carnets (assets/carnets/)
Imágenes de ejemplo de carnets de identidad:

- `carnet_mujer.jpg` - Foto de CI de mujer con texto visible "Soltera" o "Casada" (estado civil femenino)
- `carnet_hombre.jpg` - Foto de CI de hombre con texto visible "Soltero" o "Casado"

Estas imágenes se usan para simular el OCR. Pueden ser creadas con Photoshop o descargadas de ejemplos públicos.

### 3. Puntos Rojos (assets/puntos_rojos/)
Fotos de evidencia de lugares peligrosos:

- `rojo1.jpg` - Foto de calle oscura o zona peligrosa
- `rojo2.jpg` - Foto de esquina mal iluminada  
- `rojo3.jpg` - Foto de área con reportes de acoso

### 4. Pánico (assets/panico/)
Videos y audios simulados de incidentes:

- `panico1.mp4` - Video simulado de 30 segundos (puede ser video de calle desde celular)
- `panico1_audio.mp3` - Audio simulado de incidente (puede ser audio ambiente o grabación de prueba)
- `panico2_audio.mp3` - Otro audio de ejemplo

### Nota para desarrolladores:
Mientras no tengas las imágenes reales, puedes usar placeholders o imágenes temporales.
La app funcionará con cualquier imagen que coloques en estas rutas.
